xargs

发表于 2020-10-25 分类于 linux

简介

xargs命令的作用，是将标准输入转为命令行参数，通常用于管道命令。因为大多数命令都不接受标准输入作为参数，所以需要xargs。

xargs格式

xargs后面的命令默认是echo

参数

-d 更换分隔符（默认将换行符和空格作为分隔符）
-p 打印出需要执行的命令，询问用户是否要执行
-t 打印出要执行的命令，直接执行，不需要交互确认
-o 用null当作分隔符
-l 指定多少行作为一个命令行参数
-n 指定每次将多少项，作为命令行参数
-I 指定每一项命令行参数的代替字符串

–max-procs 默认只用一个进程执行命令，0表示不限制进程数；
